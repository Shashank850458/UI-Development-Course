Javascript class Object


Class : The class is collections of member variables and member methods.

Member Methods: All the user defined functions of class are called member method/function

Member Variables: All the variables defined under constructor using this operator. the member variables can be access anywhere inside/outside of the class.

Constructor: the constructor is a pre-defined function of javascript class. It is also called life cycle function of javascript class. It get call automatically while object creation of a class.

object of a class: It is instance of class.

this:The operator is called object of current class. -it work only inside the class.
